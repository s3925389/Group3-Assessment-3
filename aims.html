Aims


Our ultimate goal for this project is to create a simple informational chatbot that is able to receive a query from a user and respond appropriately. The chatbot should also be able to scale upwards when further intents are provided to the chatbot which it then can be trained on. The intents that this chatbot will be trained on will initially be mainly information related to the Conqueror team profiles and the chatbot creation process. As an extra feature, we will attempt to incorporate a function whereby users are able to request a joke from the chatbot, following which the chatbot will respond with a joke complete with the answer.

SPECIFIC GOALS:

Create chatbot:
We will firstly attempt to create the skeleton of the chatbot with a basic user interface. The frontend design will require HTML, CSS and JavaScript coding while the backend will require knowledge of Python programming and Rasa NLU/ Rasa Core.


Train + Test chatbot
This chatbot will be trained using Rasa NLU and Core, where a new machine learning based model is created each time the chatbot is provided with training data. Developers are able to test these progressive models to ensure that the chatbot has incorporated the new intents and training data and is able to understand the intent of the user and provide the appropriate response.

Upload to server and/or host on website
Finally, we will attempt to upload the chatbot to an AWS server and hopefully be able to link the chatbot with the GitHub-hosted webpage, so that once the chatbot has been trained on some limited data, a user is able to interact with the chatbot and hopefully will receive appropriate answers to their queries.
