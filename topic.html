Topic

The concept of a chatbot was based on a question posed by Alan Turing: would a person be able to distinguish between conversation with a human or a computer program? This simple question has led to research into areas such as natural language processing, machine learning and artificial intelligence, eventually culminating in the development of the first chatbot, ELIZA.


As progress has been made in each of these areas, the technology has now progressed to such a state that chatbots are now able to perform more sophisticated functions, and is now a widespread piece of technology that has evolved from primarily performing textâ€‘based interactions with humans to the creation of smart devices such as Alexa and Google Assistant that takes input from and responds to speech.


A chatbot is extremely versatile, so that one can be adapted for any type of industry. For this project, we will attempt to create a simple chatbot capable of receiving prompts from a user and supplying the relevant information in response. We will attempt to link this chatbot to a website that contains personal profiles on each of the team members involved in the project, as well as an outline of the process that was used to create the actual chatbot. Given the tone that we propose to give the chatbot in its responses, as an added feature, we will attempt to create a joke function where users are able to request hearing a joke, such that if the option is chosen, a joke is offered and the users are able to respond with an answer.
