Scope and Limits


Following a group review and analysis of the original project plan, which upon reflection appeared lacking in specificity, given the time limits and negligible start-up funding, we have decided to scale back the project in terms of setting smaller and more defined goals staged in a progressive manner such that at the end of the first five-week sprint, we will have developed a chatbot that is able to receive specific input from a user limited to requests for information on either a team member or itself, or the request for a joke, recognise the intent of the user and respond appropriately by creating a chatbot using a combination of HTML, CSS, and JavaScript coding, Rasa NLU and Core training software and deep learning processes through Keras and TensorFlow.
Given the limited time available and given the basic IT skills that the team as a whole possesses at this point in time, we believe that we will be unable to host the chatbot on a cloud server such as AWS to enable individuals to interact with the chatbot through either a standalone application or integrated onto the group website. However, this is something that with further experimentation and experience, we hope to be able to achieve in the future.
For this project, at this point our ultimate aim is to have created a chatbot that will return appropriate answers to questions in regards to team member profiles, a basic introduction to the chatbot itself or provide a joke with an answer when requested.
