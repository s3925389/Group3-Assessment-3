Testing


Testing chatbots is quite an involved and detailed process given the fact that chatbots require continuous monitoring and training as the chatbot interacts with a wider audience using different phraseology and syntax. For this chatbot, we believe we will have succeeded at creating a functioning chatbot when:


  • The chatbot is able to ask for inputs from users
  • When the input is given, the chatbot is able to analyse the input to recognise key words
  • The chatbot provides an appropriate response.

The Rasa NLU training software allows the developer to add intents for the chatbot. Once the intents are added, the “rasa train” command initiates training. Each time this command is run, a new model is generated where the developer can test the chatbot to ensure that the chatbot can respond to the new intents with the appropriate response (Rasa, 2021).


As a result, for a small chatbot with limited training data, this method will be appropriate for testing purposes as testing is able to be conducted by the developer during the initial training stages.
